# DMandML
A library that performs data mining and machine learning tasks

## Usage

Maven coordinates:

```xml
<dependency>
  <groupId>com.github.tknudsen</groupId>
  <artifactId>dm-and-ml</artifactId>
  <version>0.0.1</version>
</dependency>
```

Current snapshot:

```xml
<dependency>
  <groupId>com.github.tknudsen</groupId>
  <artifactId>dm-and-ml</artifactId>
  <version>0.0.2-SNAPSHOT</version>
</dependency>
```
